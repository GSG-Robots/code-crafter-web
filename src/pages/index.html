Was ist Code Crafter?

Code Crafter ist eine App auf der man Kostlos das Programmieren mit Python lernen kann. Dazu lernst du schritt für schritt neue Features zu deinem eigenen Spiel hinzufügen. Mit interaktiven Elementen und unsrem Maskottchen Olli ist der ganze Prozess interessant und spaßig gestaltet.

Unser Lernbaum

Unser selbst entwickelter Lernbaum ermöglicht es jeden Nutzer sich Individuell zu entwickeln. Nach und nach können durch das erweitern des Spiels neu Lektionen freigeschaltet werden. Als Programmierer hat man somit auch die Möglichkeit sich auf sein Spezialgebiet zu spezialisieren. Der Nutzer kann dabei zwischen Spieledesign, Spielemechanik und Programmieren wählen.
